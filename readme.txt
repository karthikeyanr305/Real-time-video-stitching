To install:

pip install random
pip install imutils
pip install time
pip install matplotlib
pip install scikit-image
pip install opencv-python==3.4.13.47

Files:
main.py - to be run, it holds the functions required to execute the code
stitch.py - contains helper functions for the main code

Requirements:
 - Webcam is required to get two different video sources
 - There should be overlap in the video source for it to be stitched.
